Introduction

This page demonstrates a few common ways to use the context portal.

The context portal allows code running in different contexts (frames, pages, networks) to communicate in a manner which allows client code (Remote) to add and invoke functions on server (Portal) side. All primatives, Object values and Promises are supported as function parameters as well as return types.
